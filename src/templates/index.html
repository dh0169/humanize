Humanize - AI Among Us

Overview

Humanize - AI Among Us is a fun and engaging multiplayer game where 4-6 players are grouped in an online chat. One of the players is an AI! The goal is to work together to identify the AI before time runs out.

Features

  • 4-6 player online multiplayer game
  • AI-powered gameplay
  • Interactive communication and strategy
  • Fun & suspenseful experience

Tech Stack

  • Python 3.10+
  • LLM integration
  • Pip & Virtual Environments

Getting Started

Prerequisites

  • Python 3.10+
  • pip
  • Virtualenv

Installation

  1. Clone the repository:
    2. git clone https://github.com/your-username/humanize.git
  2. Create a virtual environment:
    3. python3 -m venv venv
  3. Activate the virtual environment:
    4. source venv/bin/activate  # macOS/Linux
    .\venv\Scripts\activate  # Windows
  4. Install dependencies:
    5. pip install -r requirements.txt

Running the Game

python3 main.py
